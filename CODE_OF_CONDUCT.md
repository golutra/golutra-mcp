# Code of Conduct

## Our Pledge

We want this project to be a respectful, technically serious, and collaborative space for everyone participating in it.

## Our Standards

Examples of behavior that contribute to a positive environment:

- giving constructive technical feedback
- assuming good intent while still challenging weak ideas
- documenting tradeoffs clearly
- respecting maintainers' time and project scope

Examples of unacceptable behavior:

- harassment, intimidation, or personal attacks
- bad-faith arguments and repeated disruption
- publishing private information without permission
- demanding unpaid support as if it were an obligation

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, commits, code, and other contributions that violate this Code of Conduct.

## Reporting

For conduct issues, contact the project maintainers through the repository hosting platform's private contact mechanisms when available.
